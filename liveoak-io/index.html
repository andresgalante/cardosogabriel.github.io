LiveOak provides a flexible and easy to use services platform, on-premise or in the cloud. You focus on developing an amazing mobile application quickly, while we take away the headaches by providing all the services you need.

LiveOak Features

  • Example Applications

    Simple and ready to run applications showcasing LiveOak features across multiple devices.

  • Storage Resources

    Flexible, scalable and high performance database with the built-in MongoDB based storage.

  • Data Editor

    View and manage your data in the management console, featuring real time updates.

  • Security

    Out of the box Single Sign On (SSO) and Identity Management (IDM) with the embedded Keycloak server.

  • Push Notifications

    Add Push Notifications to your applications with the Aerogear Unified Push Server (UPS) integration.

  • Application Clients

    Whether you are developing native mobile or web applications, we've got you covered with LiveOak.

  • Business Logic

    Extend your applications' capabilities running JavaScript inside LiveOak and using the console Code Editor.

  • Analytics

    Watch your application growth and performance with the Analytics dashboard.
